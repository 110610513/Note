# 額外補充

- LAN 簡單來說就是廣播封包可送達的區域網路

- 扁平化設計即把所有電腦放在同一個網路

- VLAN 邏輯上將區域網路切分成各個區域網路

- 迴圈的好處就是擁有容錯率，但在運作上要沒有迴圈，要做到就要用到
spanning tree

- tree就是節點到節點只有單一路線

- core aggregate access
